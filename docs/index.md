# Gazebo Harmonic Kr

Gazebo Harmonic은 로봇, 센서, 환경을 시뮬레이션하기 위한 오픈 소스 플랫폼입니다.

## 시작하기

- [설치 안내](install.md)
- [튜토리얼](tutorials.md)
- [Ignition에서 마이그레이션](migration_from_ignition.md)

## 주요 기능

- 고성능 물리 엔진
- 고급 3D 렌더링
- 다양한 센서 시뮬레이션
- ROS 2 통합
- 사실적인 환경 모델링
